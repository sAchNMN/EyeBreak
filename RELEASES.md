# EyeBreak 发布记录

## v1 - 2026-07-04

状态：已验收的基线版本。

源码基线：

* 提交：`a697c05`
* 分支：`master`
* 范围：截至自启动运行路径修复为止，已经由用户验收通过的桌面端功能。

包含功能：

* 固定间隔护眼提醒弹窗。
* 跳过、暂停、恢复、立即休息、退出流程。
* 提醒弹窗暂停按钮支持鼠标滚轮调整时长。
* 系统托盘控制菜单。
* 托盘暂停时长子菜单。
* 托盘勾选状态即时刷新。
* 开机自启开关。
* 自启动运行路径修复：开机自启时从应用目录读取 `config.json`、`app_state.json` 和图标资源。
* PyInstaller Windows 可执行文件构建流程。
* 应用、托盘、窗口、任务栏自定义图标。
* 可拖拽、可贴边自动隐藏的悬浮倒计时。
* 悬浮倒计时位置持久化。
* 悬浮倒计时隐藏时跳过界面数字刷新。
* 用户离开检测，离开期间自动延后提醒。
* 托盘设置窗口。
* 全屏应用运行时延后提醒。

构建命令：

```powershell
python -m PyInstaller build.spec
```

测试命令：

```powershell
python -m pytest -q tests -p no:cacheprovider --basetemp=.tmp\pytest
```

最近一次已验收测试结果：

* 普通沙箱权限下，测试可能在清理 `.tmp\pytest` 时遇到 `PermissionError: [WinError 5]`。
* 自启动运行路径修复后的提权完整回归通过：`73 passed in 0.35s`。
